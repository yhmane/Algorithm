## commit 참고. 정책

* package : 참여자의 성을 따서 생성해 두었음
* Class   : 성 + 문제번호
EX) Hwang1463
* 입출력 : Scanner()를 이용하거나, BufferedReader/Writer를 이용하거나 자율 
* Main() 클래스에서 실행. (Main() 클래스는 공용이므로 커밋 금지)
